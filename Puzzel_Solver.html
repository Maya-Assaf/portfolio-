Puzzel Solver

Project Title: End-to-End Automated Jigsaw Puzzle Solver

This project presents a fully automated pipeline for solving jigsaw puzzles using classical computer vision and algorithmic techniques. The system takes as input a single image of scattered puzzle pieces and outputs a reassembled puzzle image. It is designed to be efficient, accurate, and modular, making it adaptable for different puzzle types and complexities.

Key Stages:

  1. Image Preprocessing:
    • Piece Extraction: Isolates individual puzzle pieces using OpenCV thresholding, contour detection, and morphological operations.
    • Noise Removal: Filters out small artifacts based on area and shape.
    • Segmentation: Segments, crops, and saves each piece for feature extraction.
  2. Feature Representation:
    • Color Descriptors: Computes mean and histogram color representations in HSV and LAB color spaces.
    • Shape Descriptors: Analyzes contours using Fourier descriptors and curvature signatures.
    • Edge Profile: Encodes borders as 1D signals capturing concave/convex patterns.
  3. Similarity Scoring:
    • Dynamic Time Warping (DTW): Compares edge profiles for non-linear alignment.
    • Color Compatibility: Calculates weighted differences between border color histograms.
    • Combined Score: Uses a weighted combination of shape and color metrics.
  4. Puzzle Assembly:
    • Greedy Matching: Iteratively selects the most compatible piece pairs.
    • High-Level Filtering: Applies spatial constraints and rotation heuristics.
    • Final Layout: Populates a grid with matched pieces until the puzzle is reconstructed.

Technologies Used:

  • OpenCV (image processing, feature extraction)
  • NumPy (numerical computation)
  • Python (implementation)

Outcome:

The system solves moderately complex jigsaw puzzles with high accuracy and speed, providing a robust framework for extensions such as handling missing pieces, irregular shapes, or real-world scenarios.
