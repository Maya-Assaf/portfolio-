Predicting Student Dropout Causes

Project Title: Student Dropout Dataset Analysis

This project aimed to explore, analyze, and model student dropout behavior using a dataset composed of institutional records and demographic information. The primary objective was to identify the key factors contributing to student attrition, understand the underlying socio-economic and academic dynamics, and build predictive models to estimate dropout risk. Insights from this analysis were intended to support the development of early warning systems and targeted intervention strategies for at-risk students.

1. Dataset Overview:

  • Data Sources: Institutional academic records, attendance logs, demographic attributes, socio-economic indicators, and historical dropout labels.
  • Data Cleaning: Addressed missing values, inconsistent labels, outliers, and irrelevant features using pandas and numpy.
  • Feature Engineering: Derived new features such as GPA trend, absenteeism rate, course load, family income bracket, and parental education level.
  • Label Encoding & Scaling: Applied encoding for categorical features and standardization for numerical variables to prepare data for modeling.

2. Exploratory Data Analysis (EDA):

  • Descriptive Statistics: Summarized distributions of academic performance, attendance, and socio-economic factors.
  • Correlation Analysis: Used Pearson and Spearman coefficients to identify relationships between variables and dropout likelihood.
  • Visualization: Utilized Matplotlib and Seaborn to create heatmaps, boxplots, histograms, and time-series plots for trend discovery.
  • Temporal Trends: Analyzed changes in dropout rates over academic terms/years and their correlation with policy or curriculum changes.

3. Modeling and Prediction:

  • Problem Framing: Treated dropout prediction as a binary classification task.
  • Model Selection: Evaluated multiple models including Logistic Regression, Decision Trees, Random Forests, and Support Vector Machines (SVM).
  • Hyperparameter Tuning: Performed grid search and cross-validation to optimize model performance.
  • Performance Metrics: Used precision, recall, F1-score, and ROC-AUC to evaluate and compare models.
  • Feature Importance: Interpreted the most impactful variables contributing to dropout using feature importance from tree-based models.

4. Results and Insights:

  • Key Dropout Indicators: Identified low GPA trajectory, high absenteeism, low parental education, and financial hardship as major dropout predictors.
  • Predictive Accuracy: Achieved a model with high recall and F1-score, suitable for early-risk identification.
  • Actionable Recommendations: Suggested personalized intervention plans, policy modifications, and support programs based on model insights.

Technologies Used:

  • Python: For all analysis, preprocessing, and modeling.
  • pandas & numpy: For data handling and manipulation.
  • scikit-learn & scipy: For machine learning models and statistical analysis.
  • matplotlib & seaborn: For data visualization and reporting.

Outcome:

The project delivered a data-driven framework for understanding and predicting student dropout, enabling educational institutions to proactively support students through data-informed strategies and reduce attrition rates.
